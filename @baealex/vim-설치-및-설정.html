리눅스 vim은 기존의 에디터인 vi보다 향상된 편의성을 제공하는 편집기이다. 이 포스트는 레드헷 계열과 데비안 계열에서 vim을 설치하는 방법과 셋팅하는 방법에 대해서 작성될 것이다. vim의 대표적인 장점이라면 vi 에디터에서 사용할 수 없었던 →←↑↓와 backspace를 사용할 수 있어서 정말 편리하다.

#Install vim in RedHat
sudo yum install vim

#Install vim in Debian
sudo apt-get install vim

vim의 설정은 모든 사용자에게 적용하거나 현재 로그인한 사용자만 적용하게 생성할 수 있다. 이 글에선 로그인한 사용자만의 vim 설정을 완료할 것이다.

vi ~/.vimrc
# Edit .vimrc in the home directory.

  • sysntax on : 코드에 색상을 입혀준다.
  • set nu | num : 코드의 행번호를 표시한다.
  • set ts : 탭 들여쓰기의 너비를 만큼 설정한다.
  • set sw : 블록시작시 들여쓰는 너비를 만큼 설정한다.
  • set autoindent(ai) & smartindent(si)
  • set cindent : C스타일의 들여쓰기를 시행한다.
  • colo - : 터미널과 sysntax의 테마를 -로 바꾼다.

이쁘다... 필자는 네이티브 터미널에서 촬영했지만 원격에서도 정상적으로 작동한다.
