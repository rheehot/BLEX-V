Microsoft Azure을 2년동안 무료로 사용해보자!!(학생 전용)

안녕하세요!! 꿀팁전도사입니다!!!!!!

Microsoft Azure 서비스를 아시나요? Azure는 Microsoft사에서 제공하는 클라우드 서버 시스템으로 아마존의 AWS와 거의 비슷한 서비스를 제공합니다.

하지만 AWS도 그렇고 Azure 도 그렇고 학생이 사용하기에는 생각 외로 부담스러운 가격을 가지고 있습니다.

개인 과제나 포트폴리오용으로 사용하는 저렴한 서버인 B1s를 대여하면 한달에 약 10,800에 + 알파(도메인 + 추가 저장소)가 소모됩니다.

살짝 욕심부려서 서버를 B2나 A1같은걸로 대여하게되면 핸드폰요금이랑 맞먹는 수준으로 올라 갈 수도 있습니다.

고맙게도 Azure은 학교 이메일을 보유하고 있으면 체험 서비스(1년 B1s 서버 임대, P6 SSD)와 Azure 100$ 크레딧까지 주는 학생용 Azure을 제공해 줍니다!

여기서 등록하시면 됩니다. 미리 Microsoft 계정을 만들고 등록하는 것을 추천합니다. 링크 : https://azure.microsoft.com/ko-kr/free/students/

꿀팁)Azure에 처음 가입하면 한달동안 이용할 수 있는 200$ 크레딧과 1년 체험 서비스를 이용 가능합니다. 따라서 1년 이용 후 학생 서비스를 구독하면 2년동안 무료로 서비스를 이용 할 수 있습니다
