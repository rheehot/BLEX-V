About me 프로젝트 시작 / 끝

사이트 주소 : https://yoyounn18.github.io/kimyoungjo/

후술할 사정에 의해서 어제 글을 못쓰고 오늘 하루에 About me 프로젝트를 끝냈다. SPA로 잘 구성해서 완성하고 만족스러운 마음으로 Git Page로의 배포를 시도했는데 왜인지 안되더라. 그렇게 한 6-7시간을 씨름한 끝에 내가 GG를 선언했다. 결국 만들어 놓았던 React router들을 모두 해제하고 한 페이지에 component들을 모아서 진짜 '싱글페이지'가 되어 버렸다. 이번 프로젝트의 성과를 정리해보자면,

  • component를 최대한 나누려고 노력하였다. (실제로도 많이 나눔)
  • css animation을 처음으로 다뤄보았다. 생각보다 단순해서 앞으로 요긴하게 써먹을 수 있을것 같다.
  • 물론 보여주진 못하겠지만 react router를 좀 더 잘 다루게 된 것 같다. 아마 다음 프로젝트에서 잘 보여질 듯 하다.

해결해야 할 부분으로는,

  • 저번 fan page를 만들었을때는 Git page를 이용해서 잘 배포가 되었는데 이번에는 무슨 이유인지 배포가 되지 않았다. 정확히는 배포는 됐는데 blank 화면이 떴다. (404 page 아님) 어차피 언제까지나 계속 Git page에 기생할 수도 없는 것이고, AWS라던가 Heroku 같은 새 배포 방법을 찾아봐야겠다.
  • 여전히 react router 에 관해 배워야 할 부분이 많다. 현재 Velopert님의 3가지 강의중 1가지만 수강한 상태인데, 그 뒷 강의들도 수강해서 더 폭넓은 활용에 힘써야 할 듯 하다.

어제 글을 못 쓴 이유

큰 사정은 아니고 어제 모종의 일로 시골에 내려가야 할 일이 생겨서 부득이하게 코딩도 못하고 글도 올리지 못했다. 그래도 긍정적인 부분은 드디어 낮과 밤을 바꾸게 되었다는 점! 사실 그동안 정말 힘들었던게 기상 하면 밤 11시고 낮에 잠들고 이러다 보니까 날짜가 헷갈려서 1일 1알고리즘 스터디나 1일 1포스팅 같은 할 일들을 했는지가 헷갈렸는데 이번에 성공적으로 밤낮을 바꿔서 그 문제는 더 이상 없을 것 같다. 이제 슬슬 반환점이 눈 앞인데 나태해지지 않는것이 가장 중요할 듯 하다.
