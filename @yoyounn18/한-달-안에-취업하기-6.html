오늘의 영상 : VELOPERT 최초 공개?! React계의 문익점이 말하는 프론트엔드 개발

React의 아버지 Velopert님의 인터뷰 영상이 있길래 가져와봤다.

전체적으로 React의 장점을 나열해 놓은 영상이다. 특히 개발자 입장에서 정말 쓰기 편하다는 말이 공감이 갔다. 개발자 입장에서의 UX가 정말 잘 되어있다고 할까? 그리고 React 자체의 문법을 배우기 보다는 기존 Javscript를 이용해서 개발하기 때문에, 스타트만 잘 끊으면 Javscript야 워낙 많이들 사용하는 언어라서 사용법이나 자료를 익히는데에 문제가 없다는 말도 보탰다.

React관련 홈페이지를 운영을 할 때 국내에서 React가 많이 쓰이길 바라셨다고 한다. 영상에서는 아니라고 하셨지만 지금의 (그때보다는) 넓어진 React 시장에서 70%이상은 공헌도가 있지 않으신가 하는 생각이 든다.

이 분의 React 강의를 많이 들어봤다. 사실상 내 React 실력의 90% 이상은 Velopert님이 만들어주셨다고 해도 과언이 아니다. 이분의 대표 컨텐츠 Velog 라는 개발자 친화적 커뮤니티가 있는데 곧 Version 2를 공개한다고 한다. (이 영상이 11월 말에 공개되었는데 올해말 내년초 라고 하니 곧 인것 같다.) 기존의 한국 최대 커뮤니티인 OKKY는 너무 Back-End 그 중에서도 Spring의 비중이 높아 보여서 얻을 수 있는 정보가 한정되어 있었는데, 이 Velog는 개발자분께서 Front-End를 다루고 계셔서 그런지는 몰라도 구글에 Front-End 관련 정보를 검색했을때 한국 정보들은 거의다 Velog에 작성된 글들 이었다. 꼭 번창하셨으면 좋겠고, OKKY를 뛰어넘어주셨으면 좋겠다.
